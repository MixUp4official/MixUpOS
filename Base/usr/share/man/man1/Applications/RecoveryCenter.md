## Name

RecoveryCenter - MixUpOS updates, diagnostics, and recovery hub

## Synopsis

```**sh
$ RecoveryCenter
```

## Description

Recovery Center is a quick access tool for common recovery and maintenance tasks in MixUpOS.
It can:

- open Software Updates
- open App Store
- open System Monitor
- open Terminal
- open the home folder
- copy a small diagnostics summary
- reboot or shut down the system
